# 💈 Barber Sync

> 🚧 Projeto em desenvolvimento

O **Barber Sync** é uma aplicação Fullstack em desenvolvimento, criada com o objetivo de construir um sistema moderno de gerenciamento para barbearias.

O projeto está atualmente em fase inicial, com foco na configuração do ambiente, definição da arquitetura e desenvolvimento das bases do frontend e backend.

Este projeto faz parte da minha prática em desenvolvimento Fullstack, aplicando conceitos de construção de interfaces, criação de APIs e organização de aplicações.

---

## 🛠️ Tecnologias

- React
- TypeScript
- Tailwind CSS
- Node.js
- Express
